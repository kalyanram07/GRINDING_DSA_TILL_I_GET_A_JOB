what is an array ??

an array is the collection of elements of same datatype stored in contiguous memory locations 
the memory representation in arrays can be done as -
index :  0   1   2   3   4 
value :  10  20  30  40  50

why do we need arrays ??
without arrays we need to store elements like

int b = 20;............
like this for small set of integers it looks simple but for storing large set of integers like storing 1000 elements !! 
so we are storing those 1000 elements under 1 name liike this ' int arr[1000];' by this we can store simply .
